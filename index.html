How to connect to GitHub (free, 3 mins)
  1. Make sure your Quest Board repo exists on GitHub
  2. Go to github.com → Settings → Developer settings
  3. Click Personal access tokens → Tokens (classic) → Generate new token
  4. Give it a name like "Quest Board", set expiry to No expiration
  5. Tick the repo checkbox (full repo access)
  6. Click Generate token and copy it immediately
  7. Enter your repo name (e.g. ryan/questboard) and paste the token
  8. Click Connect — your data saves as data.json in the repo
